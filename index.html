Politique de Confidentialité - Ilios Pub

Dernière mise à jour : 10 juillet 2025

1. Introduction

Bienvenue dans la politique de confidentialité de Ilios Pub, une application permettant aux utilisateurs de créer et diffuser des campagnes publicitaires sur des écrans publics dans différentes villes. Cette politique explique comment nous collectons, utilisons, partageons et protégeons les informations personnelles des utilisateurs. En utilisant Ilios Pub, vous acceptez les pratiques décrites dans cette politique.

2. Données que nous collectons

Pour fournir nos services, nous collectons les types de données suivants :

  • Données personnelles : Nom, adresse e-mail.
  • Données de contenu : Médias (photos, vidéos) ou textes téléversés pour les campagnes publicitaires.
  • Données de paiement : Informations nécessaires pour traiter les paiements via carte bancaire, Orange Money, MTN Money, ou autres services de paiement (par exemple, identifiants de transaction, mais pas les données complètes de carte bancaire, qui sont gérées par nos partenaires de paiement).
  • Données d'utilisation : Informations sur la manière dont vous utilisez l'application, telles que les campagnes créées, les horaires de diffusion sélectionnés, et le temps passé sur l'application.
  • Données techniques : Adresse IP, type d'appareil, système d'exploitation, identifiants uniques de l'appareil (par exemple, UDID).
  • Données de localisation : Données de localisation approximatives ou précises (si vous autorisez l'accès) pour sélectionner les écrans dans une ville spécifique.

3. Comment nous utilisons vos données

Nous utilisons vos données pour :

  • Permettre la création, la gestion et la diffusion de vos campagnes publicitaires sur les écrans sélectionnés.
  • Traiter les paiements via nos partenaires (Orange Money, MTN Money, ou processeurs de cartes bancaires).
  • Personnaliser votre expérience utilisateur, par exemple en suggérant des écrans à proximité.
  • Envoyer des notifications ou des mises à jour sur vos campagnes (si vous y avez consenti).
  • Analyser l'utilisation de l'application pour améliorer nos services.
  • Respecter nos obligations légales, y compris les exigences fiscales ou réglementaires liées aux paiements.

4. Partage des données

Nous pouvons partager vos données personnelles avec :

  • Prestataires de services tiers : Services d’analyse (par exemple, Google Analytics) pour comprendre l’utilisation de l’application, et processeurs de paiement (par exemple, Orange Money, MTN Money, ou Stripe) pour traiter les transactions.
  • Partenaires d’affichage : Gestionnaires des écrans publicitaires pour coordonner la diffusion de vos campagnes.
  • Autorités légales : Si requis par la loi, par exemple pour des audits fiscaux ou des enquêtes judiciaires.

Nous ne vendons ni ne louons vos données personnelles à des tiers à des fins commerciales.

5. Sécurité des données

Nous mettons en place des mesures de sécurité techniques et organisationnelles (comme le chiffrement des données sensibles et des protocoles sécurisés) pour protéger vos données personnelles contre tout accès non autorisé, perte ou altération. Cependant, aucune méthode de transmission sur Internet ou de stockage électronique n’est sécurisée à 100 %.

6. Droits des utilisateurs

Conformément aux lois applicables (par exemple, le RGPD pour les résidents de l’UE), vous avez les droits suivants :

  • Accès : Demander une copie des données que nous détenons sur vous.
  • Rectification : Corriger les données inexactes ou incomplètes.
  • Suppression : Demander la suppression de vos données personnelles, sauf si la loi exige leur conservation (par exemple, pour des registres fiscaux).
  • Opposition : Vous opposer au traitement de vos données pour certaines finalités, comme les notifications marketing.
  • Portabilité : Recevoir vos données dans un format structuré et lisible.

Pour exercer ces droits, contactez-nous à ilios@gmail.com.

7. Conservation des données

Nous conservons vos données personnelles uniquement aussi longtemps que nécessaire pour gérer vos campagnes publicitaires et respecter nos obligations légales (par exemple, conservation des données de paiement pour des audits fiscaux). Les médias téléversés (photos, vidéos) peuvent être supprimés après la fin de la campagne, sauf si vous demandez leur conservation.

8. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre pour refléter les changements dans nos pratiques ou les exigences légales. Toute modification sera publiée sur cette page avec une date de mise à jour actualisée. Nous vous encourageons à consulter régulièrement cette page.
